# Премиальный лендинг онлайн-курса

Luxury / high-ticket лендинг с тёмным фоном, золотыми градиентами и glassmorphism.

## Технологии

- Next.js 14 (App Router)
- TailwindCSS
- Framer Motion

## Запуск

```bash
npm install
npm run dev
```

Открой [http://localhost:3000](http://localhost:3000) в браузере.

## Структура

- `src/components/sections/` — блоки лендинга (Hero, Benefits, Program, Reviews, Pricing, Urgency, Footer)
- `src/components/ui/` — переиспользуемые UI-компоненты (GoldButton)
